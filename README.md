# face-generation-from-sketches
Conditional GAN model to generate realistic face images from sketch inputs using paired datasets.

This project implements a Conditional GAN (cGAN) to generate realistic human face images from input sketches. 
The model learns a mapping between sketch images and real faces using paired data from the Person Face Sketches dataset.

Key Features:
- Conditional GAN architecture (Generator + Discriminator)
- Sketch-to-image translation
- Training with paired dataset
- Evaluation using visual quality comparison

Tech Stack:
Python, PyTorch/TensorFlow, OpenCV, NumPy, Matplotlib

Applications:
- Forensics (suspect reconstruction)
- Digital art & design
- Face synthesis research
